## 概要
吃音者の仕事体験談共有サイト

##思い、問題意識
私自身も吃音の当事者ですが、仕事と吃音の関係は、人生の中で特に不安や苦しさを感じてきた部分でもあります。
社会人になることに不安を抱える人、また実際に働く中で悩む人が、他の吃音者の働き方を知ることで、「こんな風に働くことができるんだ」と少しでも安心感や希望を持てるようになればと思います。
